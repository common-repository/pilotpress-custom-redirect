This plugin helps you add custom redirect after login based on membership level.

== Description ==

This plugin require pilotpress plugin and can add functionality to redirect users to custom page after login based on membership level.
== Installation ==

1. Upload `pilotpress-custom-redirect` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
